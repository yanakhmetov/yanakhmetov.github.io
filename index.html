Чат

Онлайн-чат на основе WebSocket. Проект представляет собой простой чат в реальном времени. Реализованна форма для уникальности имен в чате перед тем, как в него добавиться. Чтобы пользователям было удобно, добавлены светлая и темная темы сайта. Серверная часть проекта построена на базе Express с использование библиотеки Socket.IO и отвечает за обработку новых подключений и разрывов соединения, рассылку сообщений всем активным пользователям, хранение истории сообщений в памяти сервера. Клиентская часть в свою очередь осуществляет подключение к WebSocket-серверу, позволяет делать отправку сообщений в реальном времени, отображает ленту сообщений с автоматической прокруткой вниз, показывает список подключенных пользователей. Сайт адаптирован под мобильные, планшетные и десктопные устройста. Проет написан с помощью библиотеке React на функциональных компонентах с использованием хуков UseState и UseEffect. Для хранение временных данных использовался LocalStorage.

Технологии: HTML, CSS, JavaScript, React + Vite, WebSocket, Express

Виджеты

Одностраничное веб-приложение (SPA), представляющее собой дашборд (информационную панель) с тремя интерактивными виджетами. Проект демонстрирует работу с внешними API, управление состоянием, асинхронные запросы, автодополнение (autocomplete) и адаптивный дизайн. Виджет Погода (WeatherWidget) использует API - OpenWeatherMap и отображает температуру, описание, ветер, влажность, давление, видимость. Реализован поиск погоды по названию города. Есть автодополнение городов через Geocoding API (при вводе от 2 символов). Используются Debounce-запросы (800 мс) для оптимизации. Виджет Криптовалюты (CryptoWidget) использует API - CoinGecko и отображенает курсы Bitcoin, Ethereum, Litecoin в USD. Показывает изменения цены за 24 часа (зелёный/красный цвет, стрелки). Автообновление происходит каждые 2 минуты. Виджет Поиск фильмов (MovieSearchWidget) использует API - OMDB (Open Movie Database) и отображает список фильмов. Поиск фильмов происходит по названию, в качестве стартового запроса стоит "Harry Potter". Присутствует горизонтальный слайдер с карточками фильмов и автодополнение с постерами фильмов. Используются Debounce-запросы 500 мс для основного поиска, 300 мс для подсказок. Весь проект реализован на React c использованием функциональных компонетов и хуков useState, useEffect, useRef.

Технологии: HTML, CSS, JavaScript, React, REST API, Axios
